Conditional Rendering 
  Conditional rendering of content & styling (Ternary Operator, &&, && with map, Enum)
 

  You can execute code when variable value is set or style is applied
  
      example:        const {name, isAuthenticated} = user;      //name is set to false


  <p>Welcome, {isAuthenticated ? name : "Please log in"}</p>
  Conditional rendering is repeating the same condition multiple times
  it is always the same as className, as <p> content </p> 

  example:
    <p>
      Welcome, {isAuthenticated ? name : "Please log in"}                      // (set <p> text)
    </p>             

    __________________________________________________________________

    <p className={ isAuthenticated ? 'welcomeTextAuthed' : 'welcomeText'} >     // set the style on condition 
      Welcome, {isAuthenticated ? name : "Please log in"}                       // set <p> text on same condition
    </p>
