Best practices using Form and useState 
1. Create an object as default values for the form 
const defaultData = {
  title: '',
  body: ''
}

2. send the object as value inside the useState 
const [formData, setFormData] = useState(defaultData)

3. destructuring the object to use the variables
const {title, body} = formData;

4. Create a function to set/change the state 
const onChange = (e) => {
  setFormData( (prevState) => ({
  ...prevState,
  [e.target.id]: e.target.value
  }))
};
